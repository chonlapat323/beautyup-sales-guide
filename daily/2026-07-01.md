## งานวันนี้

### API — แก้ Google Sheets 403 Permission Denied

- วิเคราะห์ error: `google.service.ts` อ่าน sheet ID จาก `process.env.GOOGLE_SHEET_ID` (ว่างเปล่า) แทนจาก DB
- แก้ `visits.service.ts` ให้อ่าน `visit_sheet_id` จาก Setting table ก่อน
- เพิ่ม step-by-step logging ใน `appendToSheetById` เพื่อ pinpoint ว่า read หรือ write ล้ม
- พบสาเหตุจริง: service account share เป็น Viewer ไม่ใช่ Editor → แก้ permission แล้วใช้งานได้
- Push: `31b8177`, `b4b9f33`, `850cf19` → `line-api main`

### API — เพิ่ม default limit

- เปลี่ยน default limit visits API จาก 20 → 100
- Push: `206d5b7` → `line-api main`

### Admin Web — แก้ visits page default period

- เปลี่ยน default จาก "ทั้งหมด" → "วันนี้" เพื่อให้เห็น visit ล่าสุดทันที
- Push: `cd8e096` → `line-admin main`

### Mobile — เพิ่ม date filter หน้าประวัติ

- เพิ่ม filter chips วันนี้ / เดือนนี้ / ทั้งหมด (default = เดือนนี้)
- แก้ `api.getVisits()` รับ `dateFrom`/`dateTo` params
- Push: `ea23270` → `line-mobile main`
