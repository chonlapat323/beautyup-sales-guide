## สถิติเซล

อันดับและรายละเอียดการเยี่ยมของแต่ละเซล

1. ดู **Leaderboard** อันดับเซลที่เยี่ยมร้านมากที่สุดในช่วงที่เลือก
2. **ค้นหาเซล** ด้วยชื่อหรืออีเมลได้ทันที
3. กดที่แถวของเซลเพื่อดู **รายการเยี่ยมร้านทั้งหมด** พร้อมกรองผล ทริป จังหวัดได้

> กรองตามช่วงเวลาและจังหวัดร่วมกันเพื่อดูประสิทธิภาพของแต่ละทีมในแต่ละพื้นที่

---

## การเยี่ยมร้าน

รายการบันทึกการเยี่ยมทั้งหมด

1. กรองรายการด้วย Dropdown (จังหวัด / ผล / ทริป / ภารกิจ / ลูกค้า)
2. กดที่รายการเพื่อดูรายละเอียด: รูปภาพร้าน, สลิป, พิกัด GPS, เลขอ้างอิง, ยอดสั่งซื้อ
3. กดรูปภาพในรายละเอียดเพื่อดูแบบ **fullscreen** พร้อม swipe และ zoom ได้

## อนุมัติสลิป

ตรวจสอบและอนุมัติสลิปการชำระเงินจากเซล

**ขั้นตอนการอนุมัติ**

1. เลือก Filter **รออนุมัติ** เพื่อดูสลิปที่ยังไม่ได้ตรวจสอบ
2. กดดูรูปสลิป ตรวจสอบยอดเงินและข้อมูลร้านค้า
3. กรอก **ยอดเงิน (บาท)** ที่ยืนยันจากสลิป
4. กด **อนุมัติ** หรือ **ปฏิเสธ** พร้อมระบุเหตุผล

**สถานะสลิป**

| สถานะ | ความหมาย |
|-------|----------|
| รออนุมัติ | เซลแนบสลิปแล้ว รอ Admin ตรวจสอบ |
| QR ยืนยันแล้ว | ระบบ Slip2Go/EasySlip ตรวจสอบผ่านอัตโนมัติ |
| อนุมัติแล้ว | Admin อนุมัติเรียบร้อย |
| ปฏิเสธ | Admin ปฏิเสธ พร้อมเหตุผล |

> **หมายเหตุ:** สลิปที่ระบบตรวจ QR ผ่านแล้วจะข้ามขั้นตอนนี้อัตโนมัติ ไม่ต้องอนุมัติซ้ำ

## ค่าคอม

คำนวณและบันทึกการจ่ายค่าคอมมิชชันรายเดือน

1. เลือก **เดือน** ที่ต้องการดู
2. ดูยอดขายและค่าคอมของแต่ละเซล (อัตรา % ตั้งค่าได้ที่เมนูตั้งค่า)
3. เซลที่ถึงเกณฑ์ขั้นต่ำจะแสดงปุ่ม **บันทึกการจ่าย**
4. แนบสลิปการโอนเงิน แล้วกด **บันทึก**

**แท็บในหน้านี้**

| แท็บ | เนื้อหา |
|------|---------|
| คำนวณ | รายการเซลพร้อมยอดขาย ค่าคอม และสถานะการจ่าย |
| ประวัติ | รายการจ่ายค่าคอมที่บันทึกไว้แล้วทั้งหมด |

> **หมายเหตุ:** เซลที่ไม่มีข้อมูลบัญชีธนาคารจะแสดงคำเตือน ให้ติดต่อเซลให้อัปเดตข้อมูลในแอปก่อน

## ตั้งค่าระบบ

กำหนดค่า API และพารามิเตอร์ระบบ

| หัวข้อ | รายละเอียด |
|--------|-----------|
| Slip Verification | เลือกผู้ให้บริการ Slip2Go หรือ EasySlip พร้อม Secret Key |
| ค่าคอม (%) | อัตราค่าคอมมิชชัน เช่น 3% ของยอดขายรวม |
| เกณฑ์ขั้นต่ำ (บาท) | ยอดขายขั้นต่ำที่ต้องถึงก่อนได้รับค่าคอม |
| Google Sheets ID | Sheet สำหรับ sync ข้อมูลการเยี่ยมและค่าคอม |

> **สำคัญ:** กด **บันทึก** ทุกครั้งหลังแก้ไข — การตั้งค่าจะไม่บันทึกอัตโนมัติ

## จัดการผู้ใช้

เพิ่ม แก้ไข และกำหนดสิทธิ์ผู้ใช้งาน

**เพิ่มผู้ใช้ใหม่**

1. กดปุ่ม **เพิ่มผู้ใช้**
2. กรอกชื่อ, อีเมล, รหัสผ่านตั้งต้น
3. เลือก **Role** (สิทธิ์) ที่ต้องการ
4. กด **บันทึก**

**การแก้ไขและลบ**

- กดไอคอน **แก้ไข** เพื่อเปลี่ยนชื่อ อีเมล Role หรือรหัสผ่าน
- กดไอคอน **ลบ** เพื่อลบผู้ใช้ออกจากระบบ (ไม่สามารถกู้คืนได้)

> แนะนำให้กำหนด Role ให้ผู้ใช้ทุกคน — ผู้ใช้ที่ไม่มี Role จะเข้าถึงระบบไม่ได้

---

## จัดการสิทธิ์ (Role)

กำหนดว่าแต่ละ Role เข้าถึงเมนูใดได้บ้าง

**ระดับสิทธิ์ต่อเมนู**

| สิทธิ์ | ทำได้ |
|--------|-------|
| ดู (View) | เข้าหน้านั้นได้ มองเห็นข้อมูล แต่ไม่มีปุ่มดำเนินการ |
| แก้ไข (Edit) | ใช้ปุ่มอนุมัติ บันทึก ส่ง สร้าง แก้ไขได้ |
| ลบ (Delete) | ลบข้อมูลได้ |

**สร้าง Role ใหม่**

1. กดปุ่ม **สร้าง Role**
2. ตั้งชื่อ Role เช่น "ผู้จัดการ" หรือ "เซลอาวุโส"
3. เลือก ดู / แก้ไข / ลบ สำหรับแต่ละเมนู
4. กด **บันทึก** แล้วไปกำหนด Role นี้ให้ผู้ใช้ที่หน้าจัดการผู้ใช้

> **ข้อควรระวัง:** Role **แอดมิน** มีสิทธิ์ทุกอย่าง ให้เฉพาะผู้ดูแลระบบเท่านั้น
